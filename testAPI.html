Получение списка мероприятий

URL метода API: http://f0578562.xsph.ru/api/event/getList.php

Данный метод возвращет массив мероприятий в формате JSON

Поля JSON:

  • name - Название мероприятия
  • description - Описание мероприятия
  • address - Адрес мероприятия
  • photo_url - URL фото мероприятия
  • subject - Тег предметной области мероприятия. Например, футбол, музыка...
  • level - Тег уровня вхождения в предметную область. Например, смотреть, читать, выступать, тренироваться...
